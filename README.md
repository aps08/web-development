# web-development

## This web project contains 6 mini-projects, made with the help of HTML, CSS and JS.
## How to run these projects:
### Step 1: Download the project you like from the file in the repository.
### Step 2: run the index.html file. *( make sure that the css, image, sounds or js files are on the same directory level)*
